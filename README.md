# Amazon question

Rotate array clockwise or anticlockwise:

**Input**:
`
    let arr = [
        [1, 2, 3],
        [4, 5, 6],
        [7, 8, 9]
    ];
`

**Output** (*Clockwise*):
`
    let arr = [
        [7, 4, 1],
        [8, 5, 2],
        [9, 6, 3]
    ];
`

**Output** (*Anticlockwise*):
`
    let arr = [
        [3, 6, 9],
        [2, 5, 8],
        [1, 4, 7]
    ];
`
